# Job Portal – Frontend Project

## Project Overview
This is a **mini Job Portal application** built using **HTML, CSS, and JavaScript**.  
The project demonstrates a complete frontend flow with **login, home dashboard, job details, and profile page**, along with **animations, responsive design, and smooth navigation**.  

It is designed to showcase **strong UI/UX, clean reusable components, and production-ready implementation**.

---

## Features

- **Login Page**  
  - Demo credentials for testing
  - Clean, animated card layout
  - Responsive design

- **Home Dashboard**  
  - Searchable job listings
  - Job cards with hover animations
  - Header with profile link
  - Fixed back button for easy navigation

- **Job Details Page**  
  - Shows job title, experience, location, and apply option
  - Fixed back button to return to home
  - Fully responsive layout

- **Profile Page**  
  - Displays user information
  - Logout functionality
  - Fixed back button to return to home

- **Animations & UI Enhancements**  
  - Gradient background animation
  - Card slide-in animations
  - Button hover effects
  - Responsive for desktop, tablet, and mobile
